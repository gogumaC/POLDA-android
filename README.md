**Cooperation** | GitHub, Slack, Figma


## 🌼 Features

- ***UI구현**🖼️*
    - 앨범 내부 화면
    - 폴라로이드 조회 화면
    - 폴라로이드 만들기 화면
- ***폴라로이드 만들기*** 기능📸
    - 갤러리에서 **사진 가져오기** 📥
    - **스티커** 붙이기 💗
    - 메모 및 글꼴 변경 ✏️
- 폴라로이드를 갤러리에 **저장**💾 **/ 공유**📤
